# FX Trade Support

毎朝のルール確認、今日の相場分析、エントリー前チェック、トレード日誌をまとめたFXトレードサポートWebアプリです。

[アプリを開く](https://toufx6367.github.io/fx-trade-support/)

## 機能

- 毎朝のルール確認クイズ
- 監視通貨ごとの相場分析チェックリスト
- エントリー前チェック
- トレード日誌
- ルールの追加・削除
- スマホ対応

## 開発

```bash
npm install
npm run dev
```

## 公開

`main` ブランチにpushすると、GitHub Actionsでビルドされ、GitHub Pagesに公開されます。
